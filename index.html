YouTube Comment Toxicity Analyzer

In my YouTube Comment Toxicity Analyzer project, I developed a Flask-based web application that analyzes YouTube comments for toxic content using machine learning. The project integrates with the YouTube API to retrieve comments and predict toxicity levels, leveraging a multi-output neural network classifier. I also incorporated interactive data visualizations, including bar graphs, pie charts, and a metrics heatmap, to present the analysis results clearly. This project provided valuable hands-on experience in web development, API integration, and machine learning, while enabling users to better understand the toxicity level of comments on YouTube videos.
